Learning Management System (LMS)
A simple web-based Learning Management System built using Python Flask and SQLite. This project allows users to register, login, view their profile, and track progress. It is suitable for college projects, internships, and GitHub portfolios.

Features:
User Authentication (Register / Login / Logout)
Profile Management
Progress Tracking
SQLite Database
Session-based Authentication
Web-based Interface using Flask & Jinja2

Tech Stack
Backend: Python (Flask)
Frontend: HTML, CSS (Jinja2 Templates)
Database: SQLite
Version Control: Git & GitHub

Run the Application - python app.py
Open in Browser - http://127.0.0.1:5000/
